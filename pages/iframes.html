Testing Scenarios

Here are some test cases to automate:

  • Context Switch: Switch the driver's context to `frame1` to interact with its content.
  • Interact Inside Frame: Click the button inside the frame and verify the text changes.
  • Return to Parent: Switch back to the main document (default content) to interact with the next frame or main page.
